Build the main home screen in `app/(main)/index.tsx`.

This is the heart of the app. User sees their weekly meal plan and daily macro progress.

DESIGN:
- Background: #FAFAFA
- ScrollView, full screen
- Bottom tab bar with 3 tabs: Home (this screen), Log, Profile

LAYOUT (top to bottom):

1. HEADER
- Left: "Good morning! 🦝" (greeting changes by time of day)
  - 5am-11am: "Good morning!"
  - 11am-5pm: "Good afternoon!"
  - 5pm-10pm: "Good evening!"
  - 10pm-5am: "Still up? 🦝"
- Right: small Rocky emoji button that shows a random fun fact when tapped
- Below greeting: user's name or "Champion" if no name set

2. FIRST LAUNCH POPUP (Modal)
Show this modal automatically if no weekly plan exists in AsyncStorage:
- Dark semi-transparent overlay
- White card centered on screen
- Large Rocky 🦝 emoji at top
- Title: "Ready to build your first meal plan?"
- Subtitle: "I'll create a personalized 7-day plan based on your goals 🎯"
- Two buttons:
  - Primary: "Let's go! 🦝" — #FF6B35, white text, full width
  - Secondary: "Maybe later" — text only, #2D2D2D, below primary
- On "Let's go!": close modal, set isGenerating to true, call generateWeeklyPlan() from services/gemma.ts
- On "Maybe later": close modal, show empty state

3. DAILY MACRO SUMMARY CARD
- Title: "Today's Progress"
- White card, subtle shadow
- 4 macro rings in a row (use MacroRing component):
  - Calories 🔥
  - Protein 🥩
  - Carbs 🍚
  - Fat 🥑
- Each ring shows: current / target
- Below rings: "X kcal remaining" or "X kcal over" in small text
- For now show 0 / target values (real logging comes later)
- Read targets from AsyncStorage key "calculationResults"

4. WEEKLY PLAN SECTION
- Title: "This Week" + small refresh icon button (top right of section)
- 7 day cards in a vertical list
- Today's card is highlighted with #FF6B35 left border and #FFF3EE background
- Other days: white background, subtle shadow

Each day card shows:
- Day name (Monday, Tuesday etc.) + date
- "Today" badge on current day (orange pill)
- 3 meal rows: Breakfast, Lunch, Dinner
- Each meal row: meal emoji + meal name + calories
- Total daily calories at bottom right
- Tapping a day card expands it (accordion style) to show full meal details

LOADING STATE:
When generateWeeklyPlan() is running:
- Replace weekly plan section with:
  - Animated Rocky 🦝 (use simple opacity animation, pulsing)
  - Text: "Rocky is cooking your plan... 🍳"
  - Subtitle: "This takes about 30 seconds"
  - Indeterminate progress bar in #FF6B35

EMPTY STATE:
When user tapped "Maybe later":
- Show a card with Rocky looking sad (🦝)
- Text: "No plan yet!"
- Button: "Generate My Plan 🦝" in #FF6B35
- On tap: show the popup again

5. QUICK ACTION BUTTON
- Floating button bottom right (above tab bar)
- #FF6B35 circle, white "+" icon
- On tap: navigate to log screen
- This is always visible
