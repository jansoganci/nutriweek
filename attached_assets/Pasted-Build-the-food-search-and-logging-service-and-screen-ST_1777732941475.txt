STEP 1 — Create `services/usda.ts`:

USDA FoodData Central API integration.
Base URL: https://api.nal.usda.gov/fdc/v1
API Key: [buraya kendi API key'ini yaz]

Implement these functions:

1. searchFoods(query: string): Promise<FoodSearchResult[]>
   GET /foods/search?query={query}&api_key={KEY}&pageSize=20&dataType=Foundation,SR Legacy
   
   Parse response and return array of:
   {
     fdcId: number,
     description: string,
     calories: number,      // nutrientId 1008
     protein: number,       // nutrientId 1003
     carbs: number,         // nutrientId 1005
     fat: number,           // nutrientId 1004
     servingSize: number,   // per 100g default
   }

   Nutrient ID mapping:
   - 1008 = Energy (kcal)
   - 1003 = Protein (g)
   - 1004 = Total Fat (g)
   - 1005 = Carbohydrate (g)

   All values are per 100g. We will calculate based on user portion input.

2. calculateNutrients(food: FoodSearchResult, grams: number): MacroResult
   Formula: (value / 100) * grams
   Returns: { calories, protein, carbs, fat } for given gram amount
